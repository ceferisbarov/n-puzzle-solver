# N-Puzzle Solver

## How to run

On Ubuntu:
```sh
# Clone the repo
git clone https://github.com/ceferisbarov/n-puzzle-solver

cd n-puzzle-solver

# Provide the input and output paths
python3 src/solve.py --input data/3.txt --output out/3.txt
```
